## Getting Started

To get a local copy up and running follow these simple example steps.


### Installation

1. Clone the repo

```sh
git clone https://github.com/mensatalk/mensatalk_app.git
```

2. Install NPM packages

```sh
npm install
```

### Usage

Run on android

```
npx react-native run-android
```

Run on iOS
```
npx react-native run-ios
```
